# CabBooking – MERN Stack Cab Booking System

This is a full-stack cab booking web application built using the MERN stack.

## Technologies Used
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- User authentication (Login / Signup)
- Book cab
- View ride history
- Responsive user interface

## Description
This project demonstrates a complete cab booking system where users can register, login, and book rides. It uses MongoDB for database, Express and Node.js for backend, and React for frontend.
